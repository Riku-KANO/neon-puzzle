# Neon Puzzle

A neon circuit puzzle rendered in 3D. Rotate blocks to route power from the source to the target. Levels are generated by the Rust/WASM core and rendered with React Three Fiber.

</p>

## Features

- 3D neon board with animated camera rig and cinematic menu.
- Difficulty selection (Easy / Normal / Hard) with size scaling.
- Electric shader overlay for energized paths.
- Celebration effects on puzzle completion.

## Quick Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Rust/WASM Core (optional)

If you change the Rust core in `core/`, rebuild the WebAssembly package into `pkg/`:

```bash
wasm-pack build --target web --out-dir pkg
```
