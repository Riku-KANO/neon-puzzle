# Gameplay Guide

## Controls

- Click a block to rotate it 90Â° clockwise.
- Route power from the source (magenta) to the target (green).

## Flow

1. **Menu**
   - Choose a difficulty.
   - Start the system to generate a puzzle.
2. **Play**
   - Rotate blocks until the target is energized.
3. **Win**
   - A celebration effect triggers.
   - Options:
     - `NEW PUZZLE` (same difficulty)
     - Select a new difficulty for another puzzle
     - `RETURN TO MENU`

## Difficulty

- **Easy**: smaller board for quick puzzles.
- **Normal**: default board size and complexity.
- **Hard**: larger board for longer routes.
